# Glm

Чат с GLM: откройте **`index.html`** локально или по ссылке после включения GitHub Pages.

## Ссылка (GitHub Pages)

1. В репозитории: **Settings → Pages**.
2. **Build and deployment** → Source: **GitHub Actions** (не «Deploy from a branch»).
3. После следующего push в `main` workflow **Deploy GitHub Pages** соберёт сайт и выдаст URL в вкладке **Actions** (и в **Settings → Pages**).

Адрес будет вида **`https://<user>.github.io/Glm/`** (для проекта из организации — `https://<org>.github.io/Glm/`). Главная страница — `index.html` в корне артефакта.
