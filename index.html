Чат с Z.AI GLM

Работает через Puter.js (модель Z.AI GLM, без API-ключа на стороне разработчика). На стороннем сайте (GitHub Pages) обычно нужно один раз нажать Войти через Puter — иначе всплывающее окно входа может быть заблокировано браузером. Использование оплачивается по модели User-Pays со стороны вашего аккаунта Puter.
